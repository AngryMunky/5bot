# Lawson Design - Claude Code Plugins

A marketplace of internal Claude Code plugins.

## Plugins

- **five-bot** - a disciplined Product -> UX -> Architect -> Developer -> QA workflow with Markdown handoffs and human approval gates. See [`five-bot/README.md`](five-bot/README.md).

## Install

```
/plugin marketplace add AngryMunky/5bot-plugin
/plugin install five-bot@lawson-design
```

Then in any project: run `/5bot-init` once, then `/product` to begin. Full loop: `/product` -> `/ux` -> `/architect` -> `/dev` -> `/qa`, with `/handoff` and `/gate` along the way.
